Comparação de Algoritmos de Ordenação

  Aqui estão alguns dos algoritmos de ordenação que eu desenvolvi durante a minha graduação. Alguns deles foram feitos para disciplinas, outros para competições de programação e outros por hobby.

  Os algoritmos estão em Java e Python, porém para este site, eu os converti para JavaScript, e para melhor visualização, eu utilizei a biblioteca p5.js para desenhar os algoritmos de ordenação. O código de todas as versões dos algoritmos de ordenação está disponível no meu GitHub, onde você pode ver o código fonte, baixar e executar os algoritmos (Vale ressaltar de ler a documentação).
