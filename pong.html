Pong

Pong foi um projeto que eu fiz em Java para a disciplina de Programação Orientada a Objetos. O jogo é uma simulação do clássico Pong, onde dois jogadores controlam barras que rebatem uma bola. O jogo foi feito puramente em Java e sem muitas bibliotecas (Maioria das funções externas foram feitas pelo professor da disciplina).
